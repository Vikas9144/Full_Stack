Q.1) What is jQuery?

jQuery is a fast, small, and feature-rich JavaScript library designed to simplify client-side scripting of HTML. It was created by John Resig in 2006 and has since become one of the most widely used JavaScript libraries.

jQuery simplifies many common tasks in web development, such as:

1) DOM manipulation: jQuery provides easy-to-use methods for selecting and manipulating DOM elements. This makes tasks like adding, removing, or modifying HTML elements much simpler and more concise.

2) Event handling: jQuery allows developers to easily attach event handlers to DOM elements, such as click, hover, or submit events, without needing to worry about cross-browser compatibility issues.

3) AJAX: jQuery simplifies asynchronous HTTP requests (AJAX) by providing a set of methods for making requests to a server and handling the responses, allowing for dynamic loading of content without refreshing the entire page.

4) Animations: jQuery includes built-in animation effects and methods for creating animations, transitions, and visual effects on DOM elements, making it easier to create dynamic and interactive user interfaces.

5) Utilities: jQuery provides various utility functions for tasks like iterating over arrays or objects, manipulating strings, and working with data.
